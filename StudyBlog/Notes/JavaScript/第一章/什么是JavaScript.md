# 什么是 JavaScript

JavaScript 是一门用于网页交互的脚本语言。作为 Web 的核心技术之一，它由以下组成部分构建：

- **ECMAScript**：语言核心 [[ECMA-262规范]](https://www.ecma-international.org/publications-and-standards/standards/ecma-262/)
- **DOM**：文档对象模型 [[MDN DOM文档]](https://developer.mozilla.org/zh-CN/docs/Web/API/Document_Object_Model)
- **BOM**：浏览器对象模型 [[MDN BOM文档]](https://developer.mozilla.org/zh-CN/docs/Web/API/Window)
