# 期末学生作业

## 项目介绍

设备全生命周期管理（Asset Lifecycle Management, ALM）是指对设备从规划、采购、运行至报废的全过程进行科学化、系统化的综合管理，覆盖设备“一生”的完整链条，旨在实现设备性能最优化、经济效益最大化及资源可持续化。

设备全生命周期管理包括设备 ‌ 规划与设计、选型采购、安装调试、运行维护、升级改造、报废处置 ‌ 等全阶段，形成从“出生”到“退休”的完整管理闭环。不仅管理设备硬件，还涵盖 ‌ 软件系统、安全协议、运行数据及附属资产 ‌，确保设备全方位可控。

## 负责模块

采购模块
用例图一：采购商务 采购专员接收来自规划模块的设备需求计划后发起寻源招标或询价流程，供应商参与投标报价，采购专员进行比价评标并最终确定供应商，随后双方签署采购合同并正式下达采购订单。

用例图二：实施交付 采购专员负责跟踪已发货订单的物流状态，实施工程师负责接收到货现场的物资并进行清点，随后执行设备的物理安装固定并进行系统上电调试与功能联调，最后发起验收移交申请由资产管理员进行最终现场验收。

活动图(采购交付全流程) 描述从接收采购需求开始经过寻源下单物流运输直到底层物理安装调试并最终验收移交的全过程链条。接收需求计划，发布招标寻源，进行评标定标，签署合同订单，供应商发货运输，现场到货接收，执行物理安装，进行调试联调，组织验收移交，完成交付。

组件图(设备物理安装) 展示采购到货的分散独立部件如何在物理现场被组装连接成一个可交付的整体系统。采购到货的独立设备部件被安装固定在指定的物理位置，通过必要的管线连接动力源与网络，并与其他相关辅助设施物理集成，最终组合构成已安装调试完成的可运行系统整体。

顺序图(验收移交) 描述当物理安装调试完成后实施工程师提交验收报告触发现场验收，验收通过后采购模块向资产模块发起新资产注册请求，资产模块完成登记并返回唯一的资产编码，标志着采购建设过程正式结束并完成向运维阶段的移交。
